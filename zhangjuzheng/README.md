《四书直解》
====

《四书直解》（原名《四书集注直解》）是明代两朝帝师、万历首辅（宰相）张居正同翰林院讲官专为万历皇帝量身定做的宫内读本。而万历皇帝即位时只有十岁，张居正用《四书》教育皇帝，要让万历皇帝做一个象唐太宗那样开明的君主。讲经同时融入了张居正等人对历代王朝兴衰的总结，以及对重大历史教训的独特见解和锐意进取的改革思想。
